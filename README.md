# CRUD Operations with JavaScript and MongoDB

## Overview
This project demonstrates CRUD (Create, Read, Update, Delete) operations using JavaScript with MongoDB. It showcases how to interact with a MongoDB database using JavaScript and provides a solid foundation for building applications that require database interactions.

## Features
- **Create**: Add new records to the database.
- **Read**: Fetch and display records from the database.
- **Update**: Modify existing records.
- **Delete**: Remove records from the database.
- **Express Framework**: Utilizing Express.js for building a server.
- **Mongoose**: Using Mongoose for MongoDB object modeling.

## Tech Stack
- **JavaScript**: The primary programming language.
- **Node.js**: JavaScript runtime for executing server-side code.
- **Express**: Web application framework for Node.js.
- **MongoDB**: NoSQL database used for storing data.
- **Mongoose**: ODM (Object Data Modeling) library for MongoDB and Node.js.

## Setup Guide
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Raghav20230/CrudJsUsingMongodb.git
   cd CrudJsUsingMongodb
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up MongoDB**:
   - Ensure you have MongoDB installed and running locally, or use a cloud-based MongoDB service.
   - Update your connection string in `config/database.js` if necessary.

4. **Run the server**:
   ```bash
   npm start
   ```
   The server should be running on `http://localhost:3000`.

5. **Test the CRUD operations** using Postman or any API testing tool.
